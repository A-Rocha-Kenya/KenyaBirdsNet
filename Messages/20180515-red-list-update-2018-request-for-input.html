Dear colleague

 

Sorry for any cross-posting but just to let you know that the BirdLife Red List team has now completed posting topics for the 2018 global Red List updated on the forum website, and are now seeking input to the discussions regarding potential Red List status changes.

 I’ve listed below the species found in Kenya that are up for discussion on the forums (click on hyperlink for more information):

Northern Ground-hornbill Bucorvus abyssinicus - request for information

Turner's Eremomela Eremomela turneri - Red List Implications of the use of Minimum Convex Polygons

White-winged Apalis Apalis chariessa- Red List Implications of the use of Minimum Convex Polygons

White-eyed Gull Larus leucophthalmus - revise global status?

Williams's Lark Mirafra williamsi -- revise global status?

Tawny Eagle Aquila rapax- revise global status?


 The preliminary deadline for comments is 30 June, so if anyone wishes to comment on the topics they should try to do so before this initial deadline. The BirdLife Red List team will then post up a draft list of preliminary proposals and there will be a period for further comments before final recommendations to IUCN are colated. The new and revised species assessments and updated factsheets will be published on the BirdLife website and incorporated into the 2018 IUCN Red List, currently scheduled for release in mid November.

 To maximise feedback and get the best possible discussion, the team would be really grateful if you could please let relevant contacts within your networks know as quickly and widely as possible.

 Finally, just to give a heads up that the Red List Team now has a new team email redlistteam@birdlife.org, which they’ll be using for any general queries/messages, so if you receive any message from that address it is from the team!

 Best wishes and thanks very much for your help, from the Red List Team
