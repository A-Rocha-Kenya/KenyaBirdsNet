Hi all
 
This sort of feedback from Marian in Singida is really important. We can never hope to monitor flamingo breeding success or otherwise by overflying Natron. All we have done is talk about this for the past decade or so.
 
In the 60s Leslie Brown monitored breeding by counting flocks and estimating the % of flamingos. If we all did this we might get a better feel for recruitment rate / breeding success.
 
BUT
 
we need counts from many lakes, ALL of them if possible. even estimates would provide useful information. 20% imms or half imms, that sort of thing.
 
many thanks
 
Neil
